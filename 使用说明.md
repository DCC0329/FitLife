# FitLife 使用说明

## 更新 App（每次改完代码后）

1. 手机用数据线连上电脑
2. 打开 Xcode → `FitLife.xcodeproj`
3. 左上角设备选 **Chanchan**（你的手机）
4. 点 **▶ 运行**
5. 等 1-2 分钟自动安装完成

> 历史数据全部保留，放心更新。

---

## 证书 7 天过期问题

- 免费 Apple ID 签名的 App **每 7 天会过期**，过期后 App 打开闪退
- **数据不会丢失**，重新用 Xcode ▶ 安装一次即可恢复
- 建议每隔 5-6 天更新一次，避免 App 失效

## 常见问题

**手机没出现在设备列表？**
- 检查数据线是否插好（需要支持数据传输的线）
- 手机弹出"信任此电脑"时点信任

**App 打开闪退？**
- 证书过期了，重新 Xcode ▶ 安装一次即可

**更新后数据消失了？**
- 不会消失，Bundle ID 不变数据就在
